--Read Me Documentation--

	-After the application is downloaded it can be opened with pythons IDLE.
	-Open the entertainment_center.py file from the movies folder.
	-Run the application by pressing F5 on the keyboard or selecting Run from the IDLE Menu then Run Module from the Run Menu.
	-A browser will open displaying Moive poster images and title.
	-Hovering over an image will highlight it.  Clicking on an image will play the trailer for that movie.
